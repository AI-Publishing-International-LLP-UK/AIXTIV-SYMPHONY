# AIXTIV SYMPHONY ACADEMY

## Project Overview

This repository contains the consolidated Academy project for the AIXTIV SYMPHONY system. It integrates several components including:

- Pilots Lounge for agent management
- Visualization Center for immersive experiences
- Gift Shop for digital assets
- Dr. Memoria's knowledge management services
- Security integration with SallyPort
- Backend administration and API services

## Directory Structure

- `/frontend`: User interface components
  - `/pilots-lounge`: Agent management dashboard
  - `/visualization-center`: Immersive visualization experiences

- `/backend`: Server-side components
  - `/admin`: Administration controls
  - `/security`: Authentication and authorization
  - `/integration`: Integration with other services

- `/services`: Specialized services
  - `/dr-memoria`: Knowledge management
  - `/gift-shop`: Digital asset marketplace

- `/courses`: Educational content
  - `/curriculum`: Course structure
  - `/content`: Learning materials
  - `/assessments`: Evaluation tools

- `/docs`: Documentation

## Getting Started

1. Install dependencies: `npm install`
2. Configure the application: Copy the template files from `/config` and set your environment variables
3. Start the development server: `npm run dev`

## Integration Points

The Academy connects with:

- Integration Gateway for cross-system communication
- SallyPort for security authentication
- Wing for pilot operations

## Project History

This repository represents a consolidation of previously separate components into a unified Academy structure, bringing together elements from various AIXTIV SYMPHONY subsystems.
