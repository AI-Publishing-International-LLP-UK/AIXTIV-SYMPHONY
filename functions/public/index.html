Warp Drive API

Welcome to the Warp Drive API. This is an API endpoint powered by Firebase Cloud Functions.

Available Endpoints

Hello World

A simple greeting endpoint to test the API.

URL: /helloWorld

Method: GET

Example: https://api-for-warp-drive.web.app/helloWorld

Update User Preferences

Update a user's warp drive preferences.

URL: /updateUserPreferences

Method: POST

Headers: Content-Type: application/json

Body Example:

{
  "userId": "user123",
  "preferences": {
    "warpSpeed": 9.5,
    "destination": "Alpha Centauri",
    "safetyProtocols": true
  }
}

Example: https://api-for-warp-drive.web.app/updateUserPreferences

API Usage Notes

This API is for demonstration purposes. All requests are processed in the us-west1 region.

For more information, please refer to the API documentation or contact the Warp Drive engineering team.
