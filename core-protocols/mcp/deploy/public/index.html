DrClaude.live MCP Server

Status: Running

This is the MCP (Model Context Protocol) server for the ASOOS system. It provides a WebSocket interface for Claude Code to connect to.

Connection Information

To connect Claude Code to this MCP server, use the following command:

/mcp add SuperClaudeX wss://drclaude.live oauth2

Available Capabilities

  • S2DO Verification
  • Co-Pilot Simulation
  • Data Access Validation
  • Domain Management
  • Test Reporting

Status Endpoint

To check server status, visit: /health
